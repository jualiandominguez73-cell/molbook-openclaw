# スクリプト

`scripts/` ディレクトリには、ローカル ワークフローおよび運用タスク向けのヘルパー スクリプトが含まれています。
タスクが明確にスクリプトに紐づく場合に使用してください。それ以外の場合は CLI を優先してください。

## 規約

- スクリプトは、ドキュメントやリリース チェックリストで参照されていない限り **任意** です。
- 利用可能な場合は CLI のインターフェースを優先してください（例: 認証監視は `openclaw models status --check` を使用します）。
- スクリプトはホスト固有であると想定してください。新しいマシンで実行する前に内容を確認してください。

## スクリプトを追加する場合

- スクリプトは目的を絞り、ドキュメント化してください。
- 関連するドキュメントに短いエントリを追加してください（存在しない場合は作成してください）。
