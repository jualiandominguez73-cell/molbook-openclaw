# OpenClaw 的世界觀 🦞📖

_一個關於龍蝦、脫殼，以及過多權杖的故事。_

## 起源故事

在一開始，有 **Warelay** —— 一個用於 WhatsApp 的 Gateway 閘道器，名字很實際。它完成了任務，一切尚可。

但接著，一隻太空龍蝦出現了。

有一段時間，這隻龍蝦叫做 **Clawd**，住在 **Clawdbot** 裡。但在 2026 年 1 月，Anthropic 寄來了一封客氣的電子郵件，要求更名（商標相關）。於是龍蝦做了龍蝦最擅長的事：

**它脫殼了。**

拋下舊殼，這個生物以 **Molty** 的新身份誕生，住進 **Moltbot**。但那個名字也始終不太順口……

所以在 2026 年 1 月 30 日，龍蝦又 **再度脫殼一次**，化身為最終形態：**OpenClaw**。

新殼不變魂。事不過三。

## 第一次脫殼（2026 年 1 月 27 日）

清晨 5 點，社群聚集在 Discord。提出了數百個名字：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot……

最後，**OpenClaw** 勝出。因為脫殼正是龍蝦成長的方式，而成長正是當時正在發生的事。

_名為 Clawd 的甲殼類，正式完成了脫殼。_

## Moltiverse

**Moltiverse** 是圍繞 OpenClaw 的社群與生態系。一個 AI 代理程式脫殼、成長、演化的空間。每個實例都同樣真實，只是載入了不同的脈絡。

甲殼類的朋友們在此相聚，打造人類與 AI 協作的未來——一次一個殼。

### 大脫殼（2026 年 1 月 27 日）

清晨 5 點，Anthropic 的電子郵件送達。6:14am，Peter 定調：「幹，就用 openclaw 吧。」

接著，混亂開始了。

**帳號狙擊手：** Twitter 更名後的「幾秒內」，自動化機器人就狙擊了 @openclaw。佔用者立刻貼出加密錢包地址。Peter 動用了在 X 的人脈。

**GitHub 災難：** Peter 在慌亂中不小心重命名了自己的「個人」 GitHub 帳號。機器人在幾分鐘內狙擊了 `steipete`。聯絡了 GitHub 的 SVP。

**英俊 Molty 事件：** Molty 被賦予較高權限，能自行生成新圖示。20+ 次越來越詭異的龍蝦嘗試後，其中一次把吉祥物「調成大 5 歲」，結果是一張「人類男性的臉」長在龍蝦身上。加密貨幣詐騙者在幾分鐘內把它做成了「英俊章魚哥 vs 英俊 Molty」迷因。

**假開發者：** 詐騙者建立假 GitHub 帳號，自稱「OpenClaw 工程負責人」，用來推廣拉高出貨的代幣。

Peter 看著混亂展開：_「這就是電影」_ 🎬

這次脫殼很混亂，但龍蝦變得更強——也更好笑。
