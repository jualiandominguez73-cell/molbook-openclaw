# `clawdbrain pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
clawdbrain pairing list whatsapp
clawdbrain pairing approve whatsapp <code> --notify
```
