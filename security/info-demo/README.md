# OpenClaw Security Atlas (Info Demo)

This is a purely informational, “wow factor” web experience that visualizes major security surfaces and mitigation themes.

## Run

From the repo root:

- `pnpm security:demo:dev`
- `pnpm security:demo:build`
- `pnpm security:demo:preview`

Or directly:

- `pnpm -C SECURITY/info-demo dev`
